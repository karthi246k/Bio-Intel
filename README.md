# BIO-INTEL 🏥🤖

## An Intelligent AI Platform for Disease Prediction and Hospital Services

BIO-INTEL is an AI-powered healthcare platform that combines **Machine Learning**, **Deep Learning**, and **Hospital Service Management** into a single intelligent web application.

The platform provides:

* Disease prediction using AI models
* Doctor appointment booking
* Lab test scheduling
* AI chatbot assistance
* Secure healthcare management system

---

## Source Code Notice 

Source code is not publicly available.

# Features

##  Disease Prediction System

* Text-based disease prediction using Machine Learning models
* Image-based disease detection using CNN models
* Supports prediction for:

  * Heart Disease
  * Diabetes
  * Kidney Disease
  * Thyroid Disorder
  * Stroke
  * Liver Disease
  * Pneumonia
  * Lung Cancer
  * Cataract
  * Skin Disease

---

##  AI Chatbot Assistance

* Collects user symptoms
* Suggests appropriate medical specialists
* Provides healthcare guidance and recommendations

---

##  Doctor Appointment Booking

* Department-wise doctor listing
* Real-time slot availability
* Token generation system
* Online / Pay Later booking options

---

## Lab Test Booking

* Online pre-booking for diagnostic tests
* Slot-based scheduling system
* Token generation for lab tests

---

##  Secure Authentication

* Username & Password authentication
* Google OAuth login
* Django password hashing for security

---

## Payment Integration

* Secure payment gateway integration
* Online payment support
* Booking confirmation after payment verification

---

## Admin Management

* Manage doctors and departments
* Manage lab tests and bookings
* Schedule availability management
* Monitor users and appointments
